# 🛒 Poe2Bay

Marketplace-style website for trading items related to **Path of Exile 2**.

The project is built on **WordPress** and aims to provide a platform where players can buy, sell, and browse in-game items.

---

## 🚀 Features

- 🛍 Item marketplace for Path of Exile 2
- 🔎 Item search and filtering
- 👤 User accounts
- 📦 Item listings
- 📱 Responsive design

---

## 🛠 Tech Stack

- WordPress
- PHP
- HTML
- CSS
- JavaScript
- MySQL
